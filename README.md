# EletricSkateBoardController
Esse é um projeto de um Skate Eletrico.
Ele pode ser controlado usando um APP Android ou um controle DIY.


Nesse projeto temos:
* Diagrama das placas
* Codigos das controladoras
* Codigo do App Android
* Calculadora do projeto
